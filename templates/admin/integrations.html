Long-lived Page Access Token előállítás (wizard)
Mit csinál ez? Egy rövid (1 órás) User Tokent vesz át, kicseréli ~60 napos long-lived user tokenre, majd ebből előállít egy soha nem lejáró Page Access Token-t a Page-edhez, és tárolja titkosítva a DB-ben.

Lépések böngészőben (~5 perc):
  1. developers.facebook.com → My Apps → Az App-edet kiválasztva → App settings → Basic. Másold ki az App ID és az App Secret mezőket.
  2. Menj a Graph API Explorer-re. Válaszd ki a User Token-t MIND A 4 PERMISSIONNEL: pages_show_list, pages_manage_posts, pages_read_engagement, pages_manage_engagement (ez utóbbi a Page-saját-kommentelhez kell!).
  3. Generate Access Token → másold ki a tokent (ez a Short User Token, 1 órás).
  4. Töltsd ki itt az alábbi mezőket és Submit.
Az App Secret-et NEM mentjük el — csak a long-lived csere során használjuk egyszer.
